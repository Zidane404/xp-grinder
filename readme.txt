An Automated MEE6 XP Grinder!

Usage: 
1) Install the requirements if you haven't already - "pip3 install -r requirements.txt" in cmd of the directory

2) Enter your Discord Token into the main.py inbetween the speech marks.

3) Type " $mee [number of messages] in the channel you wish to run the script. e.g "mee$ 100". " The bot sends 1 message every 62 seconds by default.
